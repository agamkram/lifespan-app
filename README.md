# Death Clock

One-page web app that estimates death age and year from eight health inputs.

Baseline remaining years come from **2021 U.S. period life tables** (NCHS NVSR 72-12), then evidence-based adjustments for VO₂ max, BMI, smoking, blood pressure, diabetes, and diet quality.

Not medical advice — an educational, transparent estimator with the full formula shown live at the bottom.
